Sobre The Room

No game The Room, para dispositivos Android e iOS, o jogador é apresentado a um misterioso quarto cheio de cofres antigos. O objetivo do jogo é utilizar os itens disponíveis para tentar resolver os quebra-cabeças e descobrir o que está dentro dos cofres.

Depois de receber um bilhete enigmático, o jogador deve desvendar os segredos do quarto sombrio. Munido de uma chave e um monóculo, o usuário pode explorar o ambiente movendo os dedos sobre a tela.
